# Aaha Telecom – FTTH Management System

A full‑stack FTTH (Fiber To The Home) Management System.

This repository contains:
- A Spring Boot backend (MySQL based)
- A React + TypeScript frontend

The project is designed so that even a **non‑technical user** can clone the repository
and run the system by following simple step‑by‑step instructions.

## 🚀 How to Run (Summary)

1. Set up MySQL database
2. Run backend (Spring Boot)
3. Run frontend (React)
4. Open browser → http://localhost:5173

➡️ **Detailed steps are inside each folder’s README file.**
